Ranger Selbstidentifikation

Über

Diese Java-Webapplikation wurde im Laufe einer Bachelorarbeit erstellt, welche helfen soll, die Identifikation eines Cyber-physical Systems abzuwickeln. Dieses Front-End wurde mit den folgenden Frameworks und Programmen erstellt.

Bootstrap

Für die Formatierung des HTML Textes Maven: Zum schnelleren Builden und Erstellen eines jar-files zum deployen

Freemarker

zum Trennen von Code und HTML-Text, um diesen dynamisch generieren zu können

JUnit

Für Testcases

Purejavacomm

Um auf die Ports zugreifen zu können, und in weiterer Folge um auf den Ranger zuzugreifen

Maven

Zum schnelleren Builden und Erstellen eines jar-files zum deployen

slf4j

Zum Loggen von Information

SpringBoot

Für Dependency Injection und REST
